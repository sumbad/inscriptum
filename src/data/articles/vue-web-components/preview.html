Web компоненты позволяют определять новые HTML теги, которые принято называть пользовательскими элементами (custom elements). Эти теги могут быть в дальнейшем использованы непосредственно в HTML коде вашего приложения. Применив немного JS магии вы можете прямо сейчас перевести ваши Vue.js компоненты в Web компоненты, что позволит использовать их в любых web приложениях, даже вместе с React, Angular.
