Mercurius

https://mercurius.dev/

- Automatic loader integration to avoid 1 + N queries.
- mercurius-codegen, which is designed to improve the TypeScript experience using GraphQL Code Generator (https://github.com/mercurius-js/mercurius-typescript)
- Fast (https://www.nearform.com/blog/implementing-a-graphql-gateway-with-fastify/)
- Lower cost of the infrastructure (GraphQL the Simple Way, or: Don't Use Apollo)
- Fastify + Mercurius = ❤️
